# 競プロ学習日記

## フォルダ構成
- [contest](/contest/)
  - コンテストで提出したコードを記録する
  - 問題ごとにファイルを作成する
- [study](/study/)
  - 練習や勉強で書いたコードやメモを記録する
- [answer.py](/answer.py)
  - コンテスト最中のコードテストの場所
- [knowledge.txt](/knowledge.txt)
  - tipsなどを記録する
  - なんとなく、studyよりは雑にまとめる想定
- [input.txt](/input.txt)
  - コンテストのinputのサンプルをコピペする

## コードの実行
- リポジトリルートで以下を実行
```
python3 answer.py < input.txt
```
